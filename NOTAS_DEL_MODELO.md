# Modelo de ejemplo incluido

El archivo `public/models/maqueta-urbana.glb` fue generado como maqueta urbana simple para este demo.

No usa un modelo externo descargado, por lo que puedes reutilizarlo como placeholder sin depender de licencias de terceros.

Incluye elementos básicos:

- Terreno base.
- Avenida principal.
- Calle transversal.
- Banquetas.
- Plaza central.
- Ciclovía.
- Paradero.
- Área verde.
- Árboles low-poly.
- Edificios existentes.
- Edificios propuestos.
- Luminarias.
- Bancas.
- Autos simples.

Este modelo solo es para validar navegación, Cardboard, puntos de salto y estructura técnica.
